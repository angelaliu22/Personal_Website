Fundamentals of Programming and Computer Science:
Term Project

Spring 2013

Project Description

This was my term project for 15-112 Fundamentals of Programming and Computer Science. I was inspired by personalized music players and radio stations such as Spotify, Pandora, and SoundCloud. While these players have many functionalities, I always wondered what it would be like to pick similar songs based on less traditional algorithms. I wanted to pick songs based on similar beat, rhythm, harmony, pace, etc. I sought out to see if I could produce a player of such sorts as my term project for this course. The entire program was coded in Python and the GUI was rendered with TKinter.
