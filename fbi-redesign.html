fbi.gov Redesign Challenge

Spring 2015

    Toolkit:
  • Pencil and paper sketching /
  • Sketch (Bohemian Coding) /
  • HTML, CSS, JS

In the class Advanced Web Design, one of our projects was a design challenge to redesign and implement (just a static home page) of the fbi website. I used Sketch to create mockups (which were unfortunately permanently deleted due to pressing the wrong button...), and then implemented the web page. The page is responsive as well as tablet and mobile optimized.
