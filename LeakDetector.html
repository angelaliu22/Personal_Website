Project Description

Leak Detector is a software package that actively informs users about personal information leaks to third parties on a LAN. It has a back end component that captures network data using network analysis tools like tcpdump and the Bro network monitoring framework. A series of Python scripts and filters extracts various types of personal information.



There is currently no front end or GUI for this software, so I am one of the two designers who are responsible for doing all of the UI/UX Design and front end development for Leak Detector. We are still in the early stages of brainstorming and initial mockups and prototypes (in Sketch and in HTML/CSS) to user test.
